# CCSS Dashboard

A clean, technical UI to browse **CCSS (CryptoCurrency Security Standard)** aspects, controls, and requirements (including level mapping and rationale when available).

![Dashboard](docs/images/dashboard.png)

## Requirements

- Node.js 18+ (recommended 20+)
- npm / pnpm / yarn

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Open: http://localhost:3000
